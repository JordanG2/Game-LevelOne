src/ : le dossier contenant le code source de votre jeu
game/ : le package principal contenant toutes les classes du jeu
character/ : le package contenant les classes pour les personnages du jeu (joueur, ennemis, etc.)
map/ : le package contenant les classes pour la gestion de la carte (tiles, carte elle-même, etc.)
ui/ : le package contenant les classes pour l'interface utilisateur (menu principal, écran de jeu, HUD, etc.)
item/ : le package contenant les classes pour les objets utilisables dans le jeu (armes, potions, etc.)
Game.java : la classe principale qui va initialiser les différentes parties de votre jeu (la carte, les personnages, l'interface utilisateur, etc.)
res/ : le dossier contenant les ressources (images, sons, polices, etc.) utilisées dans votre jeu
build/ : le dossier contenant les fichiers de build (compilés) de votre jeu
